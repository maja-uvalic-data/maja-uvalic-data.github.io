Project Overview

This project focuses on analyzing influenza activity over time. As part of the CareerFoundry Data Analytics program, I explored seasonal patterns, peaks, and trends to better understand how influenza evolves across months and years. With my background in pharmacy, I was particularly interested in how data can support public health decisions and planning.

Objectives

  • Identify seasonal peaks in influenza activity.
  • Compare influenza levels across different time periods.
  • Visualize trends with time-series plots.
  • Explore potential relationships with external factors (e.g., temperature, time of year).

Data & Tools

The analysis is based on time-series data of influenza activity (weekly or monthly measurements), combined with additional contextual data where available. I used the following tools:

  • Pandas – loading data, cleaning, resampling, aggregation
  • Matplotlib & Seaborn – line plots, seasonal patterns
  • Jupyter Notebook – exploratory analysis and documentation
  • NumPy – basic numerical operations
