Influenza Season

Exploratory and time-series analysis of influenza activity over time, combining epidemiological data with external factors to better understand seasonal patterns.

Python Pandas Time Series Seaborn Matplotlib Healthcare
View on GitHub View on Tableau

Project Overview

The US influenza season leads to increased patient loads, especially among vulnerable populations. This project helps a medical staffing agency plan temporary staff across all 50 states by analyzing influenza trends, determining when and how many staff are needed. Excel is used for data analysis and PowerPoint for visualization, enabling proactive workforce planning for the upcoming flu season.

Objectives

  • Identify seasonal peaks in influenza activity.
  • Compare influenza levels across different time periods.
  • Visualize trends with time-series plots.
  • Explore potential relationships with external factors (e.g., temperature, time of year).

Data & Tools

The analysis is based on time-series data of influenza activity (weekly or monthly measurements), combined with additional contextual data where available. I used the following tools:

  • Pandas – loading data, cleaning, resampling, aggregation
  • Matplotlib & Seaborn – line plots, seasonal patterns
  • Jupyter Notebook – exploratory analysis and documentation
  • NumPy – basic numerical operations

Methodology

  1. Loaded the influenza dataset and parsed date/time columns.
  2. Handled missing values and ensured a clean, continuous time index.
  3. Created additional time-based features (month, year, season).
  4. Used line plots and rolling means to understand trends and seasonality.
  5. Compared different seasons/years to identify unusual patterns.

Visualizations

The main results are presented through time-series visualizations that highlight seasonal peaks, long-term trends, and year-to-year differences.

Key Insights

  • Influenza activity typically peaks during the winter months, with a clear seasonal pattern that repeats across years.
  • Transitional periods between autumn and winter often show a rapid increase in activity, which can be critical for planning vaccination campaigns.
  • Time-series visualizations make it easier to communicate risks and expectations to non-technical stakeholders in healthcare.

Conclusion & Reflection

This project allowed me to combine my pharmacy background with data analytics skills by working on a healthcare-related time-series problem. Understanding seasonal patterns in influenza can support better planning, resource allocation, and communication with patients and healthcare professionals. In the future, this analysis could be extended with forecasting models to predict upcoming influenza peaks.
